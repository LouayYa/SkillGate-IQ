## Overview

The Acme Corp Cloud Certification Program (ACP) defines the internal certifications engineering staff pursue to validate cloud, DevOps, data, and architecture skills. Each certification lists the skills it assesses, the recommended number of focused study hours, prerequisites, and the practice-score threshold a learner should reach before sitting the exam.

A learner is considered exam-ready only when they have (a) met or exceeded the recommended study hours and (b) reached the target practice score for that certification. See LRN-BP-005 for readiness criteria.

## ACP-CE1 — Acme Certified Cloud Engineer (Associate)

- **Level:** Associate
- **Prerequisites:** None
- **Recommended study hours:** 20
- **Target practice score before exam:** 75%
- **Skills assessed:** Cloud Compute Fundamentals; Storage & Data Services; Identity & Access Basics; Networking Essentials; Cost Management
